# Обмен с АСТУ (Frontol 6)

## Источник истины

Актуальная спецификация интеграции и формата выгрузок находится в
`docs/Frontol_6_Integration.pdf`. Этот файл — обязательный reference.

## Что держим в синхронизации

Этот документ — краткая конспективная версия и навигация по PDF. При любом
обновлении правил выгрузки, структуры транзакций или полей, сначала меняется
PDF, затем обновляется этот файл.

## Быстрая навигация по PDF

- **Выгрузка данных**: структура файла, шапка, правила повторной выгрузки.
- **Список транзакций**: перечень типов транзакций и ссылки на описания.
- **Особенности транзакций**: правила интерпретации полей и кодов.
- **Таблицы полей**: спецификация форматов по каждому типу транзакций.

## Связанные документы

- `docs/database/DATABASE.md` — описание БД и общих принципов хранения.
- `docs/database/TRANSACTION_TABLES_SPEC.md` — семантика полей.
- `docs/database/TRANSACTION_TABLES_SCHEMA.md` — схема таблиц.
